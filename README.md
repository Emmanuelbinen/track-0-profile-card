## ğŸ¨ Quick Customization

### Change Profile Info
Edit `index.html`:
```html
<h2 class="user-name">Your Name</h2>
<p class="user-bio">Your bio here</p>
```

### Change Colors
Edit `styles.css` (top of file):
```css
:root {
    --primary-red: #dc2626;    /* Your main color */
    --primary-white: #ffffff;  /* Your accent color */
}
```

**Try these colors:**
- Blue: `#2563eb`
- Green: `#16a34a`
- Purple: `#9333ea`

### Update Social Links
